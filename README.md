# 🚀 Throw-Your-Text  

**Throw-Your-Text** is a simple and quick tool for sharing text between computers without the hassle of signing up, logging in, or relying on external messaging apps. Designed for environments like computer labs, it eliminates the need to log in to apps like WhatsApp just to share a snippet of code or text.  

## 🌟 Features  
- **Instant Sharing:** Quickly share text between computers without registration.  
- **No Sign-Up Hassles:** No account creation or login required—just copy, paste, and share.  
- **Simple Interface:** Minimalistic design for fast and easy use.  
- **Secure:** Your shared text is temporary and not saved permanently.  
- **Cross-Device:** Works seamlessly on any computer with internet access.  

## 🎯 Objectives  
- Save time in environments like computer labs where logging into personal accounts isn't ideal.  
- Provide a fast, no-frills solution for text sharing without compromising privacy.  
- Avoid the dependency on messaging apps for temporary text sharing.  

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript.  
- **Backend:** Node.js with Express.js.  
- **Database:** Mongo db (for temporary text storage).  
- **Hosting:** Railway or Render for deployment.  

## 🚀 Future Plans  
- Add **QR Code Sharing** for easier text transfer.  
- Implement an **auto-delete timer** for enhanced privacy.  
- Enable **drag-and-drop support** for file sharing alongside text.  
- Add **end-to-end encryption** for secure communication.  
- Introduce a **history log** for retrieving recently shared texts.  

## 💡 Inspiration  
Throw-Your-Text was born out of the frustration of having to log in to apps like WhatsApp in a shared environment like a computer lab just to share a quick code snippet or text. This tool provides a lightweight, no-login alternative for fast and efficient sharing.
